# 📌 Project Overview

The **Heart Disease Prediction System** is a production-style Machine Learning web application designed to predict the probability of heart disease using patient medical attributes.

This project demonstrates a complete end-to-end Machine Learning workflow including:

- Data preprocessing
- Feature engineering
- Feature scaling
- Machine Learning model training
- Real-time prediction pipeline
- Interactive frontend integration
- Deployment-ready application structure

The application allows users to provide medical information through an interactive dashboard and instantly receive:

- Heart disease risk prediction
- Prediction confidence score
- Structured patient summary
- Interactive prediction feedback

# ⚡ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/heart-disease-prediction.git
```

---

## 2️⃣ Navigate to Project Directory

```bash
cd heart-disease-prediction
```

---

## 3️⃣ Create Virtual Environment

### Windows

```bash
python -m venv venv
```

---

## 4️⃣ Activate Virtual Environment

### CMD

```bash
venv\Scripts\activate
```

### PowerShell

```bash
.\venv\Scripts\Activate.ps1
```

---

## 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Application

```bash
streamlit run app.py
```

# 📊 Input Features

The model predicts heart disease risk using the following medical attributes:

| Feature | Description |
|---|---|
| Age | Patient Age |
| Sex | Gender |
| Chest Pain Type | Chest Pain Category |
| Resting Blood Pressure | Blood Pressure Level |
| Cholesterol | Cholesterol Level |
| Fasting Blood Sugar | Blood Sugar Indicator |
| Resting ECG | Electrocardiogram Result |
| Maximum Heart Rate | Peak Heart Rate |
| Exercise-Induced Angina | Exercise-Related Chest Pain |
| Oldpeak | ST Depression |
| ST Slope | Slope of ST Segment |
